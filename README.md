# qianxvde.github.io

Personal homepage of Ziqian Guo, an undergraduate at China Agricultural University working on humanoid robot soccer, reinforcement learning locomotion, motion retargeting, and robot deployment.

Visit: [https://qianxvde.github.io](https://qianxvde.github.io)

## Local preview

```bash
python3 -m http.server 8000
```

Then open `http://localhost:8000`.

## Structure

- `index.html` — homepage content
- `assets/css/theme-luka.css` — responsive light/dark theme
- `assets/img/cau_logo.png` — China Agricultural University logo
- `assets/cv/Ziqian_Guo_CV.pdf` — curriculum vitae

The site is static and can be deployed directly with GitHub Pages from the `main` branch.
